# Xporade

## How to Install
1. Git Clone this repository
2. Open the project and copy the .env.example file
3. rename the file into .env 
4. Register a new account in kolosal AI website
5. Get your api key from the web and assign the VITE_API_KEY variable with Kolosal AI API key

6. Visit firebase console and login with your account
7. Create a new project in firebase and assign a name
8. Visit project settings and add a Web App
9. Copy the firebase config and assign it to the proper variables in the env

10. run "npm i" in vscode cms
11. run "npm run dev" in vscode cmd
